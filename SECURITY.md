# Security Policy

## Supported Versions

Versions 1.0+ will receive updates and bug fixes.

| Version | Supported          |
| ------- | ------------------ |
|  1.0 +  | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

Use the [issues](https://github.com/CmdrAvegan/AmbiTuya/issues) page to report vulnerabilities or problems.
